== Description ==

iMasters WP Hacks is a solution to personalize the WordPress admin to final users. Hide some widgets, show anothers, hide menu itens and controll what your client can see.

== Installation ==

How to install iMasters WP Hacks

e.g.

1. Upload `imasters-wp-hacks` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use the plugin through the 'iMasters WP Hacks' menu in WordPress
